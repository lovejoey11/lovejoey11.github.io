Command Line Short Keys

Posted on 2017-08-14 Edited on 2021-03-10 In learning

Introduction

This blog will introducing some common short keys including fast moving cursor in command line, command editing, edit history command and execute, bang(!) command and control command.
Let’s make basher more productitive

Declearation

Ctrl - k : hold Ctrl key, then press k;
Alt - k : hold Alt key, then press k;
M - k : press ESC once, then press k;

Move curser

Ctrl – a : move cursor to the begin of current command  
Ctrl – e : move cursor to the end of current command
Ctrl – b : move cursor left one character
Ctrl – f : move cursor right one character
Alt – b : move cursor left one word
Alt – f : move cursor right one word
Ctrl – xx : move cursor between current location and begin of command
M-b : move cursor left one word
M-f : move cursor right one word

Edit current command

Ctrl – h: delete the character before(left) cursor
Ctrl – d: delete the character after(right) cursor
Ctrl – w: delete the all words before cursor from current cursor location
Alt – d : delete the words after current cursor to the end of line
M – d : delete a word from current cursor location
Ctrl – k: delete all characters from cursor location to the end of command
Ctrl – u: delete all characters form cursor location to the begin of command
Ctrl – y: paste deleted content to current cursor location
Alt + t : swap the current character with two previous characters
Alt + . : Use the laster parameter from last commmand
Ctrl – _: revert to provious status

use "Ctrl - a" + "Ctrl - k" or "Ctrl - e" + "Ctrl - u" or "Ctrl - k" + "Ctrl - u" combination will able to delete the whole line.

Execute last command

!! : get back last exectued command

!wget : execute the most recent command which starts with wget
!wget:p : only print out the most recent command which starts with wget
!$ : get the last parameter from last command same as Alt - . and $_
!* : get all the pararmeters from last command
!*:p : only print out all the parameters from last command
^abc : delete abc from last command
^abc^xyz: replace abc to xyz in the last command, then execute
^abc^xyz^: replace abc to xyz in the last command
!-n : execute last n commands; !-1 will execute last command, !-5 will execute last five commands

Search history command

Ctrl - p : show the last command of current command
Ctrl - n : show the next command of current command
Ctrl - r : searching command with in the history to maching your input string
Ctrl - g : exit from searching mode

Control command

Ctrl - l : clean the screen, only retain the current line.
Ctrl - s : stop the screen to print
Ctrl - q : allow the screen to print
Ctrl - c : stop current command
Ctrl - z : halt current command

Repeat

M - <operation times> <operation command> : eg. M - 5e will print out 5 ‘e’ in the command line

note: All commands are also compatible with macOS, excluding the short keys need alt
